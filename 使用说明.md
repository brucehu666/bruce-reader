# PCReader 文本阅读软件 使用说明

## 系统要求
- Windows 7 SP1 及以上版本
- Windows 10/11 (推荐)

## 安装方法
1. 解压 `PCReader.zip` 到任意目录
2. 直接运行 `PCReader.exe` 即可使用（绿色便携版，无需安装）

## 功能说明

### 1. 文件操作
- **打开文件**: 支持 .txt 格式文件
- **自动编码识别**: 支持 UTF-8、GBK、GB2312、GB18030 等多种编码
- **最近打开**: 快速访问最近打开过的文件
- **大文件支持**: 对超大文件进行分块加载，优化性能

### 2. 阅读设置
- **字体大小**: 点击工具栏 "A+" 和 "A-" 调节
- **阅读主题**: 支持默认、护眼、夜间、绿色等多种主题
- **自动保存**: 设置自动保存到配置文件

### 3. TTS朗读功能
- **播放/暂停/停止**: 底部控制栏操作
- **语速调节**: 0.5x - 2.0x 可调，步进 0.1x
- **音量控制**: 0 - 100% 可调
- **语音选择**: 可选择系统安装的不同语音
- **位置高亮**: 朗读时自动高亮当前句子
- **从选中位置开始**: 选中文本后点击播放从选中处开始

### 4. 书签管理
- **添加书签**: Ctrl+B 或工具栏按钮
- **跳转到书签**: 右侧书签面板双击书签
- **搜索书签**: 支持按名称搜索
- **文件夹分类**: 支持创建和管理书签文件夹

## 快捷键
- Ctrl+O: 打开文件
- Ctrl+B: 添加书签

## 数据存储
- 配置文件: 程序目录下 `data/config.json`
- 书签数据: 程序目录下 `data/bookmarks.json`
- 最近文件: 程序目录下 `data/recent_files.json`
- 所有数据存储在程序目录，不写入系统注册表和用户目录

## 注意事项
1. TTS功能需要系统安装语音引擎（Windows系统默认自带）
2. 首次打开大文件可能需要稍长时间
3. 建议定期备份书签数据文件
